# Number Guessing Game (Python)

This is a simple **Python-based Number Guessing Game** where the user tries to guess a randomly generated number between 1 and 100. After each guess, the game provides feedback to guide the player.

## Features

- Random number generation between 1 and 100
- User input validation
- Feedback after each guess: "Too high", "Too low", or "Correct"
- Tracks number of attempts taken to guess the correct number
---
## How to Run

1. Make sure Python 3 is installed on your system.
2. Clone this repository or copy the code into a file named `number_guessing_game.py`.
3. Run the game from the terminal or your Python IDE:

```bash
python number_guessing_game.py
